.. admonition:: Examples
    :class: dropdown, tip

    .. jupyter-execute::
        :hide-code:
        :hide-output:

        from quantify.utilities._docs_helpers import create_tmp_dir_from_test_dataset
        import quantify.data.handling as dh

        tuid = "20210422-104958-297-7d6034"
        tmp_path, tmp_dir = create_tmp_dir_from_test_dataset(tuid=tuid)
        dh.set_datadir(tmp_path)

    For a simple analysis:

    .. jupyter-execute::

        import quantify.analysis.base_analysis as ba
        a_obj = ba.Basic1DAnalysis(tuid=tuid).run()

    For an analysis subclass that requires passing in some arguments to the :meth:`quantify.analysis.base_analysis.BaseAnalysis.run` method:

    .. jupyter-execute::

        import quantify.analysis.ramsey_analysis as ram
        a_obj = ram.RamseyAnalysis(tuid=tuid).run_until(
            interrupt_before=ram.RamseyAnalysis.analysis_steps.STEP_3_ANALYZE_FIT_RESULTS,
            artificial_detuning=250e3, qubit_frequency=4.7149e9
        )
